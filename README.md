## 关于

> Makeit Uploader 上传组件，基于 Vue3.0 + Vite 开发的。支持单图 / 多图上传及预览，单文件 / 多文件上传，分片上传，秒传，断点续传等，能满足日常开发中对上传组件的大部分需求。

## 安装

```bash
npm i makeit-uploader
```
